Git步骤

1.初始化目录的版本控制

git init

2. 查看仓库当前状态

git status

3. 将仓库文件添加到缓存区

git add . / git add –all

4.将暂存文件提交至本地仓库

git commit -m ‘提交描述’

5.建立远程仓库连接

git remote add origin git@github.com:biaofendou/biaofendou.github.io.git

6.推送到远程仓库

git push -u origin master
